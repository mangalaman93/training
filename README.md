## Basic Git Commands

* `git clone `
* `git status`
* `git checkout -b developer`
* `git branch`
* `git add <>`
* `git commit`
* `git push origin developer`
* `git log`
* `git pull origin master`


## Basic Go Commands

* `go mod init`
* `go mod tidy`
* `go build -o app`
* `go install`
* `go run main.go`

* `docker ps`
* `docker cp`
* `docker inspect`
* `docker kill`
* `docker rm`
